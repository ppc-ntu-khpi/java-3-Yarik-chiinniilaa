# Практична робота: Пошук другого за величиною числа

Цей проект містить реалізацію алгоритму для знаходження **другого за величиною** унікального числа в масиві цілих чисел.

## Опис завдання (Варіант 14)
Необхідно знайти елемент, який є меншим за абсолютний максимум, але більшим за всі інші елементи масиву.

**Приклад:**
* Вхідний масив: `[10, 45, 2, 45, 32]`
* Максимум: `45`
* Друге за величиною: `32`



## Алгоритм роботи
1. Ініціалізуємо дві змінні: `max` та `secondMax` мінімально можливими значеннями.
2. Проходимо по кожному елементу масиву:
    * Якщо поточне число більше за `max` — старе значення `max` стає `secondMax`, а поточне стає `max`.
    * Якщо поточне число менше за `max`, але більше за `secondMax` — оновлюємо тільки `secondMax`.
3. Повертаємо `secondMax`.
